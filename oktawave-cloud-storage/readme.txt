== Description ==
OKTAWAVE OCS WORDPRESS Plugin allows you to put your media/gallery files directly onto your OCS Account in OKTAWAVE

1. Allows you to upload media files to OCS account directly from media gallery of Wordpress
2. Synchronise media gallery between OCS and media Wordpress DIR

== Installation ==
1. Upload `oktawave-ocs.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
